# Auction Credit Platform Demo

React + Vite MVP demo for an auction platform using internal credits.

## Features

- Buyer / seller / admin demo flow
- Internal Toy Credit wallet using localStorage
- Member tiers: CLASSIC, PRO, VIP
- Room levels: BASIC, PRO, VIP
- Demo bid and pending-credit logic
- Modern responsive UI with Tailwind CSS

## Run locally

```bash
npm install
npm run dev
```

Then open the local Vite URL shown in the terminal (usually `http://localhost:5173`).

## Optional production build check

```bash
npm run build
```

## Notes

This is a prototype only. It has no real backend, no real payment, and no real KYC processing.
